## 📝 文档维护

### 文档组织原则

- **guides/** - 面向操作的指南和教程
- **planning/** - 项目管理和计划文档
- **sprints/** - Sprint 历史记录
- **technical/** - 技术深度文档
- **setup/** - 配置和初始化文档

### 更新文档

当你需要更新文档时:

1. 确定文档类型,放入对应目录
2. 使用清晰的文件命名(大写字母 + 下划线)
3. 在本 README 中添加索引链接
4. 更新相关文档中的交叉引用
